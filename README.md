# Smart Vision AI - Material Classifier

An intelligent system that classifies materials as usable or non-usable and provides recommendations for their use or disposal.

## Features
- 📸 Upload material images (wood, metal, plastic, glass, etc.)
- 🤖 AI-powered classification (usable vs non-usable)
- 📊 Detailed analysis (damage score, rust detection, edge analysis)
- 💡 Smart recommendations for reuse or disposal
- 🚀 Simple web interface
- 🔄 Easy model retraining
